1. Selecting by Id (getElementById)
2. Selecting by Class (getElementsByClassName)
3. Selecting by Tag Name (getElementByTagname)
4. Selecting by Query Selector (querySelector)
5. Selecting Multiple Elements (querySelectorAll)

Question-2: What is the difference between innerHTML, innerText, and textContent ?

=>Difference Between innerHTML, innerText, and textContent in JavaScript

1. innerHTML : Returns or sets the HTML inside an element, including tags.
2. innerText : Returns or sets only visible text, ignoring hidden elements and extra spaces.
3. textContent : Returns or sets all text, including hidden elements, without parsing HTML.

Question-3: What is event delegation in the DOM?

=>Event Delegation in the DOM :

Event delegation is a technique in JavaScript where instead of adding event listeners to multiple child elements, you add a single event listener to a parent element. The event then "bubbles up" from the child elements to the parent, allowing you to handle all child elements efficiently.

Question-4: What is event bubbling in the DOM?

Event bubbling is a concept in JavaScript where an event triggered on a child element "bubbles up" to its parent elements, all the way up to the <html> element.
